Deployment Architecture

The PHA Maintenance MIS is built as a hybrid modern web application:

Static Frontend CDN (GitHub Pages / Netlify)

Serves this fast, responsive user interface, custom typography, Government branding, and interactive CSS block maps globally without server load.

Live Ubuntu Backend Server

Located at /var/www/pha-dashboard on our production server. Runs Laravel PHP controllers, SQLite database migrations, billing propagation, and attendance tracking.
